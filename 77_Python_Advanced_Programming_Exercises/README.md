# 77 Python Advanced Programming Exercises

Este directorio contiene las soluciones y notas de trabajo del libro "77 Python Advanced Programming Exercises".

## Estructura

Cada ejercicio se encuentra en un archivo individual con el siguiente formato:
- `exercise_01_nombre_descriptivo.py`
- `exercise_02_nombre_descriptivo.py`
- ...
- `exercise_77_nombre_descriptivo.py`

## Organización

Los ejercicios están numerados del 01 al 77 para mantener un orden consistente y facilitar la navegación.

## Formato de cada ejercicio

Cada archivo incluye:
- **Descripción**: Breve descripción del problema y requisitos
- **Solución propia**: Implementación de nuestra solución
- **Solución del libro**: Solución propuesta en el libro (cuando esté disponible)
- **Pruebas**: Casos de prueba y verificación de resultados
