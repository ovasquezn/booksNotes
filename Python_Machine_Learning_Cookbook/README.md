# Python Machine Learning Cookbook

Este directorio contiene las notas, ejemplos y ejercicios del libro "Python Machine Learning Cookbook".

## Estructura

El libro está organizado por capítulos, cada uno con sus respectivos notebooks y scripts:

```
Python_Machine_Learning_Cookbook/
├── README.md
├── Chapter01/
│   ├── README.md
│   ├── chap_01_*.ipynb
│   └── ...
├── Chapter02/
│   ├── README.md
│   ├── chap_02_*.ipynb
│   └── ...
├── data/
│   └── (datasets y archivos de datos)
└── utils/
    └── (utilidades y funciones auxiliares)
```

## Organización

Cada capítulo contiene:
- **README.md**: Descripción del capítulo y lista de ejemplos
- **Notebooks**: Un notebook del capítulo
- **Scripts Python**: Código adicional si es necesario

## Formato de notebooks

Cada notebook incluye:
- **Descripción**: Explicación del problema o técnica
- **Código**: Implementación paso a paso
- **Resultados**: Visualizaciones y análisis
- **Notas**: Observaciones y aprendizajes
